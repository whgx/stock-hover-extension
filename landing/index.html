功能一览

从实时行情到持仓管理，一应俱全

🖱️

悬浮查行情

在任何网页上选中股票名称或代码，自动弹出实时行情卡片。也支持右键菜单和 Alt+S 快捷键触发。

📈

迷你分时图

内置 Canvas 高清分时走势图，标注最高/最低/昨收，动态时间轴，一眼掌握当日走势。

⭐

自选股管理

一键收藏自选股，Popup 面板中实时展示所有自选股行情，涨红跌绿一目了然。

🔔

价格预警

设定涨破/跌破价格，后台自动轮询监控，触发条件后通过 Chrome 通知系统桌面推送提醒。

💰

资金流向

展示主力/超大单/大单/中单/小单资金净流入柱状图，洞察主力资金动向。

💼

持仓盈亏跟踪

记录买入价和数量，自动计算加权成本、浮动盈亏金额和百分比，全局持仓汇总。

🔗

相关个股

同板块成份股一览，快速发现同行业联动机会，点击即可跳转查看。

📰

公司公告

最新公司公告列表，点击直达东方财富公告详情页，不错过任何重要信息。

🔍

模糊搜索

Popup 内置模糊搜索，支持中文名称、代码、拼音首字母，250ms 防抖，键盘上下导航。

三步上手

不到一分钟，开始使用

1

下载安装

从 GitHub 下载源码，打开 chrome://extensions，开启「开发者模式」，点击「加载已解压的扩展程序」。

→
2

选中即查

在任何网页上选中股票名称（如「贵州茅台」），行情卡片自动弹出。或使用右键菜单、Alt+S 快捷键。

→
3

管理投资

点击工具栏图标打开 Popup，搜索股票、添加自选、设置预警、记录持仓，一站式管理你的全部投资数据。
