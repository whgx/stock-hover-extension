# 选股行情助手 v3.0

Chrome 扩展：选中网页上的股票名称或代码，即可弹出实时行情卡片。支持 A 股、港股、美股。

## 安装方法

1. 打开 `chrome://extensions/`
2. 右上角打开「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本文件夹（`stock-hover-extension`）
5. 去任意网页选中股票名称或代码即可

## 功能列表

### 核心功能 (v1.0)
| 功能 | 说明 |
|------|------|
| 🖱️ 选中即查 | 选中股票名称/代码自动弹出行情卡片 |
| 实时行情 | 现价、涨跌额、涨跌幅、今开/昨收/最高/最低/成交量/成交额/振幅 |
| 多市场支持 | A 股、港股、美股自动识别 |
| 涨红跌绿 | 中国股市惯例配色 |

### P1 增强 (v2.0)
| 功能 | 触发方式 |
|------|---------|
| 📋 右键菜单 | 选中文本 → 右键 → "查询股票行情" |
| ⌨️ 快捷键 | Alt+S 查询选中文本 |
| ⭐ 自选股收藏 | 卡片中 ★ 按钮 → Popup 查看列表 |
| 📊 迷你分时图 | 卡片底部 Canvas 走势线 + 价格坐标轴 |
| 🔔 价格预警 | 设定目标价位/涨跌幅 → 桌面通知 |
| 📌 固定卡片 | 图钉按钮，固定后不自动消失 |

### P2 深度功能 (v3.0)
| 功能 | 说明 |
|------|------|
| 💰 资金流向 | 主力/超大单/大单/中单/小单净流入，红绿柱状对比 |
| 🔗 相关个股 | 同行业板块前 5 只股票，点击切换查询 |
| 📰 公司公告 | 最新 5 条公告列表，点击打开原文 |
| 📑 Tab 切换 | 行情 / 资金 / 相关 / 公告 四个 Tab 懒加载 |

### Bug 修复 (v2.1)
- 卡片不再因微小滚动/鼠标移动而消失
- 分时图增加价格坐标轴和时间标签
- 预警类型切换时单位自动联动（元 ↔ %）

## 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 在当前标签页注入卡片 |
| `storage` | 存储自选股和预警 |
| `contextMenus` | 右键菜单 |
| `notifications` | 价格预警通知 |
| `alarms` | 定时检查预警 |
| `commands` | Alt+S 快捷键 |
| `host_permissions` | 访问东方财富行情接口 |

## 使用技巧

1. **固定卡片**：点击 📌 后卡片不会自动消失，方便长时间观察
2. **切换 Tab**：资金/相关/公告 Tab 切换时才加载数据，不影响行情查询速度
3. **相关个股**：点击相关个股行可直接切换到该股的行情
4. **Popup 搜索**：点击工具栏图标，在搜索框输入股票名称/代码可直接添加自选
5. **预警设置**：在卡片中点 🔔 设置目标价，后台每分钟自动检查
